<h1 align="center">SourceTree Tutorial</h1>
<p align="center"><img src="https://user-images.githubusercontent.com/48220015/112126886-ef0f8a00-8bd5-11eb-9dc8-531f252ce2ca.jpg"/></p>

SourceTree is a free software which offers a graphical user interface to interact with your git repositories. It simplifies use of git commands and helps managing repositories with a simple user interface. It is easy to use so it can be very helpful especially for people just started to use git. It is possible connecting both local and remote Git repositories and it also works with GitHub, GitLab etc.
